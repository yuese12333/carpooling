# 接口汇总清单

> 前后端 API 接口汇总，联调状态以当前代码为准。

---

## 一、路径命名约定

| 规则 | 说明 | 示例 |
|------|------|------|
| 统一前缀 | 所有接口使用 `/api/` | `/api/users/me` |
| 资源复数 | 集合资源用复数名词 | `/api/rides`、`/api/trips` |
| 动词禁止出现在路径中 | 用 HTTP 方法表达操作 | `POST /api/rides` |
| 子资源用斜杠 | 从属关系用路径嵌套 | `/api/profile/car/update` |
| 操作例外 | 非 CRUD 动作可用子路径动词 | `/api/auth/logout`、`/api/sms/send` |
| 连字符分隔 | 多词路径用 `-` 不用 `_` | `/api/cancel-reasons` |
